# Prompt Karşılaştırmaları

## 1. Kısa Prompt
**Prompt:** Python'da liste nasıl sıralanır?
**Çıktı:** Kod örneği verildi ama açıklama eksikti.
**Yorum:** Hızlı ama öğretici değil.

## 2. Detaylı Prompt
**Prompt:** Sen bir Python eğitmenisin... (tam prompt)
**Çıktı:** Açıklama + kod örneği + sade anlatım.
**Yorum:** Öğretici ve iyi yapılandırılmış.

## 3. Örnekli Prompt
**Prompt:** 'Ali 5 elma aldı.' örneğine benzer cümleler üret.
**Çıktı:** Anlamlı ve benzer 2 cümle üretildi.
**Yorum:** Dil öğrenimi için çok uygun.

## 4. Rol Tabanlı Prompt
**Prompt:** Sen deneyimli bir teknik editörsün... (tam prompt)
**Çıktı:** Düzgün redakte ve önerilerle geldi.
**Yorum:** Profesyonel sonuç.

## 5. Kısıtlı Prompt
**Prompt:** 100 kelimeyi geçmeyecek şekilde özetle...
**Çıktı:** Kısa, resmi ve özet bir cevap geldi.
**Yorum:** Zaman kısıtlı işler için ideal.

---

## Genel Değerlendirme
- **En etkili:** Detaylı + Rol tabanlı
- **En hızlı:** Kısa prompt
- **En kontrollü çıktı:** Kısıtlı prompt
