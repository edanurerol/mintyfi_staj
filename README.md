# Prompt Karşılaştırma Çalışması

Bu branch, farklı prompt türlerinin çıktı kalitesi üzerindeki etkilerini gözlemlemek amacıyla hazırlanmıştır.

##  Amaç

- Kısa, detaylı, örnekli, rol tabanlı ve kısıtlı prompt'ların sonuçlarını karşılaştırmak
- En etkili prompt yapısını belirlemek
- Prompt mühendisliği pratiği yapmak

##  Dosyalar

- `prompt_comp.md`: 5 farklı prompt varyasyonu ve yorumlarla birlikte karşılaştırması

##  Prompt Türleri

1. **Kısa Prompt**
2. **Detaylı Prompt**
3. **Örnekli Prompt**
4. **Rol Tabanlı Prompt**
5. **Kısıtlı Prompt**

##  Sonuç

Yapılan değerlendirmeye göre:
- **En etkili**: Detaylı ve rol tabanlı prompt'lar
- **En hızlı**: Kısa prompt
- **En kontrollü çıktı**: Kısıtlı prompt

##  Kullanım

Bu dosya, yapay zeka sistemleriyle daha etkili iletişim kurmak isteyen herkes için rehber niteliğindedir.
